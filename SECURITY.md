# Security Policy

## Supported Versions

| Version | Supported |
|----------|-----------|
| Latest | ✅ |

## Reporting a Vulnerability

If you discover a security vulnerability, please do not open a public issue.

Instead, email us with:

- Description of the vulnerability
- Steps to reproduce
- Impact assessment
- Proof of concept (if available)

We will investigate all legitimate reports and aim to respond within 72 hours.

## Scope

This repository contains documentation and utilities for the Sales Tax Calculator project.

Any vulnerabilities affecting the calculator logic or API should be reported privately.

Thank you for helping keep this project secure.
